## Project Overview
A Flask-based news portal featuring categorized news, user authentication, and an admin dashboard.


## GitHub Repository
[https://github.com/AayushGuragain99/news-web](https://github.com/AayushGuragain99/news-web)

## Installation Instructions
1. **Clone the repo:** `git clone https://github.com/AayushGuragain99/news-web.git`
2. **Create Venv:** `python -m venv venv`
3. **Activate Venv:** `venv\Scripts\activate` (Windows)
4. **Install Deps:** `pip install -r requirements.txt`
5. **Setup DB:** `python seed.py`
6. **Run:** `python run.py`

- Admin-only dashboard
